Lyringo is a command line interface (CLI) game perfect for microlearning by translating lyrics line by line.

## Features

Choose a song to translate by:
1. Letting Lyringo randomly choose from a Spotify playlists.
2. Manually search for a song title and artist name.

- Uses Spotify API to allow users to play the game with their own playlists.
- Fetches lyrics via the Genius API and translates lines using Google Translate.

Line-by-line gameplay:
- Examine the original lyrics.
- Try to translate into your specified language.
- Analyze your answer with the actual translation.
